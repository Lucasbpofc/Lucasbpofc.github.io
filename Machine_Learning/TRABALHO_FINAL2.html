Este relatório apresenta o desenvolvimento e a avaliação de modelos preditivos aplicados a um experimento com cana-de-açúcar conducted sob o Delineamento em Blocos Casualizados (DBC) na estação experimental de Oratórios, MG. O estudo avalia 60 famílias distribuídas em 4 blocos (repetições), totalizando 240 parcelas experimentais. Os dados originais foram sumarizados ao nível de média por família no arquivo dd3.xlsx, totalizando 60 entradas analíticas.

O objetivo central é praticar e comparar técnicas de aprendizado estatístico para problemas de Regressão e Classificação:

Problema de Regressão: Predizer a produtividade contínua das famílias, mensurada em Toneladas de Colmos por Hectare (TSH), visando identificar e selecionar os materiais genéticos mais produtivos antes da pesagem física massiva no campo.

Problema de Classificação: Predizer a classe de rendimento da família (TSH_class), segregada a partir da média geral do experimento, avaliando a acurácia de modelos supervisionados na triagem dicotômica.

O R^2 ajustado com 0.9043 indica que mais de 90% de toda variação da produtividade da cana de açúcar entre as 60 famílias é explicada pelas variáveis do modelo. O modelo apresentou 3 variáveis altamente significativas, SN, SD e SH, indicandoque se a planta tem mais hastes, maior altura e maiores densidades ela pesa mais. O grande problema do modelo é a falta de parcimônia, que será explorada a seguir; Outro ponto importante é que amatriz de correlação plotada anteriormente indicou variáveis altamente correlacionadas, uma forma de confirmar essa autocorrelação é através do VIF

Os elevados valores do VIF indicam uma multicolinearidade severa, esses achados conjutamente do grafico de correlação indicam que o modelo contendo todas as variáveis se mostra inadequado. Esses indicativos sugerem que devemos utilizar métodos de reduçãod e dimensionalidade para reduzir a multicolinearidade presente nos dados.

Para a construção do modelo PCR, todas as 9 covariáveis preditoras foram submetidas à Análise de Componentes Principais. Como as variáveis originais possuem diferentes unidades de medida (contagens, centímetros, porcentagens de reflectância), a padronização centralizada e escalonada (scale. = TRUE) foi rigorosamente aplicada para evitar distorções de escala.

# 1. Execução do PCA com padronização obrigatória
dados_pca_total <- dados %>% select(-Family, -TSH,)
pca_total <- prcomp(dados_pca_total, center = TRUE, scale. = TRUE)

# 2. Cálculos das Proporções de Variância
var_explicada <- pca_total$sdev^2
prop_var <- var_explicada / sum(var_explicada)
var_acumulada <- cumsum(prop_var)

# 3. Criação do Dataframe para o Gráfico
df_pca_var <- data.frame(
  Componente = factor(paste0("PC", 1:length(prop_var)), levels = paste0("PC", 1:length(prop_var))),
  Proporcao = prop_var,
  Acumulada = var_acumulada
)

# 4. Plotagem do Scree Plot com Variância Acumulada

A análise visual do gráfico orienta a escolha do número de componentes a serem retidos para o modelo de regressão. O primeiro componente (PC1) retém a maior parcela isolada da variância da amostra. Seguindo o critério empírico de retenção de informações, optou-se por utilizar os 4 primeiros componentes principais (PC1 a PC4). Juntos, eles explicam de forma acumulada aproximadamente 85% a 90% de toda a variabilidade presente nas 9 variáveis originais, garantindo que a redução de dimensionalidade elimine a multicolinearidade sem causar perdas significativas de informação preditiva.

# Estruturação do banco de dados com os 4 primeiros Componentes Principais
dados_regressao_pcr <- cbind(dados %>% select(TSH), pca_total$x[, 1:4])

# Ajuste do Modelo PCR
modelo_pcr_total <- lm(TSH ~ PC1 + PC2 + PC3 + PC4, data = dados_regressao_pcr)

A aplicação do procedimento Stepwise sobre os componentes principais confirmou a ineficiência preditiva do PC2. O modelo PCR refinado, retendo exclusivamente PC1, PC3 e PC4, manteve a acurácia global praticamente inalterada (\(R^2_{adj} = 86,79\%\)). Contudo, a remoção do componente ruído provocou um aumento substancial na Estatística F do modelo (de 99,54 para 130,3), evidenciando um ganho expressivo de robustez analítica. O Biplot gerado atesta esta dinâmica: o PC1 isola a variação espectral altamente correlacionada (VIF máximo de 27.366 no NGRDI), enquanto o PC2 mapeia a arquitetura física (SN contraposto a SH e SD), fenômeno de compensação que não reflete em acúmulo de tonelagem (TSH).
